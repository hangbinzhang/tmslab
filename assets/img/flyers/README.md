## Notes

- Export page 1 of the flyer PDF to JPG via Preview (or any PDF tool).
- Lowercase filename, underscores for separators, `.jpg` / `.jpeg` / `.png` / `.webp`.
- The `.tmslab-study-flyer` wrapper caps the displayed width to 560px and lets
  the image bleed slightly outside the card padding on mobile viewports.
