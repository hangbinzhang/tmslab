# Gallery photo naming convention

Add photos to this folder with filenames in this format:

    YYYY-MM-DD_short_description.jpg

Examples:
- `2024-06-15_lab_retreat.jpg`
- `2024-09-03_new_member_welcome.jpg`
- `2025-01-20_sfn_conference.jpg`

Rules:
- Use the actual date the photo was taken
- Use underscores `_` between words (will display as spaces)
- Lowercase letters only
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

The gallery will auto-sort by date (newest first) and display captions like
"Lab retreat" with the date "Jun 2024" shown below.

Files without a date prefix still work — they just won't have a date label
and will appear at the bottom.
